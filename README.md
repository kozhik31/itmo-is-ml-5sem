# itmo-is-ml-5sem

Репозиторий с лабораторными работами по машинному обучению (5 семестр, ИТМО).  
Каждая лабораторная — в отдельной папке, внутри: ноутбуки/код, задания и (по возможности) датасеты.

---

## 📁 Структура репозитория

- **1-parser-kozhiк31/** — сбор данных (парсер / API / предобработка датасета)
- **2-supervised-kozhiк31/** — обучение с учителем (регрессия, метрики, валидация)
- **3-unsupervised-kozhiк31/** — обучение без учителя (кластеризация, понижение размерности)
- **4-text-kozhiк31/** — NLP (векторизация, классификация, quality metrics)
- **5-image-v2-kozhiк31/** — CV (регрессия/свертки/аугментации/transfer learning)
- **7-recsys-kozhik31/** — recsys (user-item split, ALS, метрики @K, UMAP-визуализация эмбеддингов)
